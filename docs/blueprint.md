# **App Name**: Etsy Atlas Manager

## Core Features:

- User Authentication: Secure user authentication and role management (admin, user).
- Order Management: Create, read, update, and delete orders, tracking details such as etsyOrderId, orderPrice, and status.
- Product Management: Add, edit, and remove products from the catalog.
- Financial Calculations: Automatically calculate totalExpenses and profit based on orderPrice, orderCost, shippingCost and additionalFees.
- Data Visualization Dashboard: Use generative AI to build an overview of key metrics like total orders, profit margins, etc.

## Style Guidelines:

- Primary color: Deep Indigo (#F5AD18) to convey trust and sophistication.
- Background color: Very light grey (#FAF3E1) for a clean, uncluttered workspace.
- Accent color: Deep Purple (#FF6D1F) to draw attention to key interactive elements and reinforce the theme.
- Body and headline font: 'Inter' sans-serif, known for its modern, machined, objective, neutral look.
- Use minimalist line icons to represent different order statuses, product categories, and user roles.
- A clean, card-based layout that is responsive across devices.
- Subtle transitions when navigating between views, editing data and filtering options.
